Mode 'mesh'

A crucial step in the solution procedure of a given problem is the creation of a mesh. This one is the discretization of the physical domain where the process to study takes place. This means partitioning the domain (1-D, 2-D or 3-D) into specific subdomains. More commonly, intervals in 1-D, triangles or quadrilaterals in 2-D, and tetrahedra, hexahedra or pentahedra in 3-D.
IRIS does not contain a powerful mesh generator. We use a simple 2-D generator issued from the OFELI library. For 3-D meshes we recommend the use of open source mesh generators like Gmsh or Tetgen.

Once the module mesh is activated, a series of keywords are accessible:

dim <n> Set space dimension: n must be 1, 2 or 3. By default, this value is set to 2. Note that the choice of some keywords below leads automatically to a prescribed dimension.
1d Generate a 1-D mesh.
rectangle Generate triangulation of a rectangular domain
cube Generate mesh of a cube
vertex Define a vertex in the polygonal domain
line Define a (straight) line in the polygonal domain
circle Define a circle (a portion or the entire one) in the polygonal domain
subdomain Generate triangulation of a rectangular domain
generate Generate triangulation of a rectangular domain
plot Generate triangulation of a rectangular domain
clear Generate triangulation of a rectangular domain
read Generate triangulation of a rectangular domain
save Generate triangulation of a rectangular domain

Command: 1d

This command activates a submode where the following commands are available:

interval <xmin> <xmax> Set the interval that defines the domain
subdivision <ne> Define the number of elements to generate
codes <cmin> <cmax> Define codes (integers) to assign to extreme vertices: A nonzero code means that no value will be prescribed on this node
nbdof <n> Define number of degrees of freedom to associate to each node (By default: 1)
save [file] Generate mesh and save it in file. If the file name is not provided, mesh is savec in file iris-1d.m
< or end Return to main menu

Command: rectangle

This command activates a submode where the following commands are available:

x <xmin> <xmax>
y <ymin> <ymax>
ne <nx> <ny>
codes <c1> <c2> <c3> <c4>
nbdof <n>
save [file]
< or end Return to main menu

Command: cube

This command runs a submode where the following commands are available:

x <xmin> <xmax>
y <ymin> <ymax>
z <zmin> <zmax>
ne <nx> <ny> <nz>
codes <c1> <c2> <c3> <c4>
nbdof <n>
save [file]
< or end Return to main menu
